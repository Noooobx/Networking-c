
# Networking Experiments

Contains implementations of TCP, UDP, FTP, and Sliding Window Protocol.

## Experiments
1. **TCP Client-Server**
2. **UDP Client-Server**
3. **FTP Client-Server**
4. **Sliding Window Protocol**

## Usage
- Run the server before the client.
- Modify IP and ports as needed.
